{{Le site|The website}} pythontutor.com {{permet de voir un code s'exécuter ligne par ligne. |allows you to see a code running line by line.}} {{Malgré son nom, pythontutor marche aussi pour Java, C++, JS et d'autres ! |Despite its name, pythontutor also works for Java, C++, JS and others !}}

{{ Va sur cette page, écris-y ton code, sélectionne la version Python 3.6 et clique sur Visualize execution. | Go on this page, write your code, select the Python 3.6 version and click on Visualize execution. }}

{{Ensuite clique sur Forward plusieurs fois pour que le programme avance ligne par ligne. |Then click multiple times on Forward so that the programs runs one line at a time. }}

{{Tu verras l'affichage (print) du programme dans le cadre Print output. |You'll see the print of the programs in the frame Print output. }}

{{Tu vois l'état de la mémoire à droite de l'écran en dessous du texte Frames. |You see the state of the memory at the right side of the screen below the text Frames. }}
