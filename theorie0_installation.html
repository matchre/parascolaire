Bonjour !

Voici comment installer les logiciels pour le cours. La théorie de ce cours se veut très courte et va droit au but. Dans chaque chapitre de théorie, je demande de lire la partie Base. Il existe aussi une partie Pour en savoir plus qui est facultative. Pour donner un chiffre sur se veut très courte, voici le nombre de pages A4 pour chaque chapitre :

theorie1 theorie2 theorie3 Total
Base 2 1 3 6
Total 5 4 8 15

En moins de 10 pages, vous avez toute la théorie nécessaire.

Dans le cadre du parascolaire, je fais les deux premiers chapitres de théorie pendant 4h, puis on attaque la partie graphique (faire des fenêtres !) pour les 14h restantes. Le chapitre 3 de la théorie sera appris à chaque étudiant en fonction de son avancée de projet.

Pour tester un code, vous avez deux options, la première n'installe rien du tout et la deuxième vous installe tout ce qu'il vous faut (obligatoire à partir de la séance 5) !

Que vous choisissez l'option 1 ou l'option 2, je vous souhaite un Happy Coding et vous invite à commencer la théorie numéro 1 !

Option 1 : Tester en ligne

Tester votre code en ligne sur pythontutor. Choisissez votre version de python (probablement Python 3, sinon lisez la remarque voir ci-dessous). et cliquez sur exécuter, le programme s'exécute pas à pas. Appuyez sur Next pour avancer d'une ligne. Vous pouvez aussi sauter à la fin en appuyant sur End.

Un exemple ici.

Cependant dès que nous ferons des fenêtres, vous devrez installer pygame et donc faire l'Option 2.

Option 2 : Installer

Vous devrez installer :

  • python
  • pygame (pour les fenêtres)

Et pour les intéressés :

  • numpy (pour les matrices utiles en 3D)
  • opengl (pour la 3D)

Le mieux est toujours d'avoir la dernière version de python (3.6) mais sous windows l'installation de pygame est plus facile avec python 3.4

Pour installer python 3.4 et pygame sous windows, faites ceci :

  • Téléchargez python 3.4 version 32 bits ici.
  • Pygame pour python 3.4 version 32 bits ici.

Ensuite vous lancer le programme IDLE, vous faites File → New, écrivez du code, enregistrez le fichier sur votre bureau, et appuyez sur F5 (ou Run dans les menus) pour le lancer.

Pour un autre éditeur python, je conseille PyCharm, la version Community ou Education est tout à fait gratuite !

Cependant, pour pouvoir utiliser les nouveautés python, c'est mieux d'utiliser des versions plus récentes (3.5, 3.6...). Ou bien si vous êtes sous mac, c'est le même niveau de difficulté que d'installer 3.6 que 3.4

Option 3 : Installer via pip et les whl

Si vous avez un souci, envoyez moi un mail !

Téléchargez la dernière version de python (3.6) sur python.org.

Nous allons utiliser pip qui permet d'installer n'importe quel paquet python ! Ici nous allons installer le paquet pygame pour faire des fenêtres.

Une ancienne version de python existe : python 2, pour savoir les différences avec python 3, lisez la remarque ci dessous.

Windows

  • Déterminez si votre version de python est 32 bits ou 64 bits, quand vous allez dans le menu windows et tappez "python" cela devrait être marqué. Même si votre ordinateur est 64 bits, il se peut que vous ayez téléchargé python 32 bits (et ça marchera très bien) Pour savoir si votre ordi est 32 ou 64 bits, allez dans "Panneau de configuration → Système"
  • Téléchargez pygame ici, et téléchargez en fonction de votre version de python (32 bits, 64 bits) (cp36 pour Python 3.6).
  • Placez le fichier téléchargé dans le dossier (existant) qui se trouve ici (remplacez 32 par 64 si votre version de python est 64 bits) : 
    <Dossier Caché AppData>\Local\Programs\Python36-32\Scripts\
  • Pour aller dans le dossier caché AppData :
    • Allez dans C:\Users\<VotreNom> (bouton windows en bas à gauche de l'écran → Ordinateur → Utilisateur → VotreNom)
    • Activez afficher les dossiers cachés [Windows 8/10] ou [Windows 7]
    • Cliquez sur le dossier AppData !
  • Ouvrez cmd (cliquez sur le bouton windows en bas à gauche de l'écran → tappez cmd)
  • (32 bits) entrez les commandes suivantes : 
    
                cd %AppData%\..\Local\Programs\Python36-32\Scripts\
                pip.exe install pygame-1.9.2-cp36-cp36m-win32.whl
  • (64 bits) entrez les commandes suivantes : 
    
                cd %AppData%\..\Local\Programs\Python36-64\Scripts\
                pip.exe install pygame-1.9.2-cp36-cp36m-win_amd64.whl

Mac

Téléchargez la dernière version de python (3.6) sur python.org.

Ouvrez un terminal (cliquez sur l'icône de recherche et tappez Terminal ou bien suivez ceci).

Pour tester que python version 3 s'est bien installé, tappez python3 puis appuyez sur Enter.

Vous devriez voir le texte Python 3.6 affiché à l'écran.

Si vous voyez une phrase du style commande introuvable: python3, c'est qu'il s'est pas installé.

Fermez le terminal et relancez en un autre, tappez maintenant les commandes suivantes :

  • Installer XCode via l'AppStore
  • Puis lancer une commande ruby pour installer homebrew (brew). 
    
                    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • Puis lancer plein de commandes brew et pip pour installer le reste. 
    
                    brew install caskroom/cask/brew-cask
                    brew cask install xquartz
                    brew install python3
                    brew install python
                    brew linkapps python3
                    brew linkapps python
                    brew install git
                    brew install sdl sdl_image sdl_ttf portmidi libogg libvorbis
                    brew install sdl_mixer --with-libvorbis
                    brew tap homebrew/headonly
                    brew install smpeg
                    brew install mercurial
  • Et finalement, installer pygame en une seule ligne : 
    
                    pip3 install hg+http://bitbucket.org/pygame/pygame

Remarque sur les versions de python

Si vous avez reçu une clef usb avec python, la version est 2.7 (ancienne version) Les ordis à l'école ont la version récente : 3.5 (nouvelle version) Pour ce cours, ça ne change quasi rien, à part trois petites différences suivantes :

Python 2 Python 3
print a,b print(a,b)
7/2 == 3 7/2 == 3.5
La gestion des accents dans les chaînes de caractères.

Si vous avez python 2 et voulez utiliser certaines fonctionnalités de python3, ajoutez une ligne comme ceci au début du fichier : 


            from __future__ import print_function  # pour le print
            from __future__ import division        # pour la division

Pour les accents, c'est plus compliqué, mais from __future__ import unicode_litterals est un bon début
