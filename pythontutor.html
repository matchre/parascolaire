Pythontutor

Le site pythontutor.com permet de voir un code python s'exécuter ligne par ligne.

Écris-y ton code, sélectionne la version Python 3.6 et clique sur Visualize execution.

Ensuite clique sur Forward plusieurs fois pour que le programme avance ligne par ligne.

Tu verras l'affichage (print) du programme dans le cadre Print output.

Tu vois l'état de la mémoire à droite de l'écran en dessous du texte Frames.
