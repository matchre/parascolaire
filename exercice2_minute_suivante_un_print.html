Exercice 2 : La minute suivante avec un seul print

Énoncé

Comme l'exercice précédent !

Sauf que tu ne peux avoir qu'un seul print dans le code !

L'unique print du code sera print(h, m) à la fin du code. 


        h = 15
        m = 30

        # écris ton code sans print ici

        print(h, m)

Rappelle toi que les variables peuvent changer de valeur avec variable = valeur, si ton code précédent avait déjà un seul print, tu peux directement passer à l'exercice suivant...
